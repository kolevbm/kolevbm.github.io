# Aliases / ейлиъси / псевдоними за команди

## Временни псевдоними за команди

За да създадете псевдоним за **текущата сесия** на терминала:

```bash
alias ll='ls -la'
```

Този псевдоним (alias произнася се "ейлиъс") ще бъде валиден само за **текущата сесия** на терминала. След като затворите терминала, ще бъде **загубен**.

## Постоянни псевдоними за команди

За да направите `aliases` (ейлиъси / псевдомини) постоянни, трябва да ги добавите към файла `~/.bashrc`. Този файл се изпълнява всеки път, когато стартирате нова сесия на терминала.

1. Отворете файла `~/.bashrc` в текстов редактор:

   ```bash
   nano ~/.bashrc
   ```

2. Добавете вашите псевдонимни в края на файла:

   ```bash
   # Aliases
   alias ll='ls -laFh'
   alias gs='git status'
   ```

3. Запазете файла и излезте от текстовия редактор.

4. Ъпдейт на сорса с команди `~/.bashrc`:

   ```bash
   source ~/.bashrc
   ```

   Вече може да ги използвате в текущата сесия на терминала и във всички следващи сесии.
