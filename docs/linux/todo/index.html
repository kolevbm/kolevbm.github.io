Export default editor​

before exporting it as an environment variable with export it is good to know the location of the bin / app you can do it with which cmd.

  • which : this command gives you the location of the bin

example:

alpine:~$ which nano
/usr/bin/nano
alpine:~$
bash
export EDITOR=/usr/bin/nano
