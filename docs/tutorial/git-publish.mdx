# Configuration

Example configuration in `docusaurus.config.js`, where `kolevbm` is the `username` of your GitHub account

```md title="docusaurus.config.js"
// Set the production url of your site here
url: "https://kolevbm.github.io",
// Set the /<baseUrl>/ pathname under which your site is served
// For GitHub pages deployment, it is often '/<projectName>/'
baseUrl: "/",

// GitHub pages deployment config.
// If you aren't using GitHub pages, you don't need these.
projectName: "kolevbm.github.io", // Usually your repo name.
organizationName: "kolevbm", // Usually your GitHub org/user name.
deploymentBranch: "gh-pages",
trailingSlash: undefined,
```

For more in-depth settings check [Deploying to GitHub Pages](https://docusaurus.io/docs/deployment#deploying-to-github-pages)

# Publish command

```powershell title="shell command"
cmd /C 'set "GIT_USER=kolevbm" && npm run deploy'
```
