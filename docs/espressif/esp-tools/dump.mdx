# Flash region Dump

## 1. read partition table esptool region

generally it is on addres `0x8000` or `0x9000` the size is always `0xc00`

```bash title="read partition table"
 esptool.py read_flash 0xc00 ptable.img
```

## 3. Read NVS content

0x1000 (4KB), if nvs size is 16KB size is 0x4000

```bash title="read_flash"
esptool.py read_flash 0x9000 0x4000 nvs.img
```

## 4. Erase NVS content

0x1000 (4KB), if nvs size is 16KB size is 0x4000

```bash title="erase_region"
esptool.py erase_region 0x9000 0x4000
```
